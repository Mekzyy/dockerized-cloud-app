# Dockerized Cloud Application

This project demonstrates containerizing a Python Flask application using Docker.

## Technologies
- Python
- Flask
- Docker
- AWS EC2

## Run Locally
docker build -t devops-cloud-app.
docker run -p 5001:5000 devops-cloud-app
